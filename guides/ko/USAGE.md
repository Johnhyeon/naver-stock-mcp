## 좋은 질문의 3요소

```
❌ 나쁜 질문: "삼성전자 어때?"
✅ 좋은 질문: "삼성전자 120일 일봉 보고 20일 이동평균선 기준으로
              추세 판단하고, 수급도 같이 봐서 매수 관점 의견 줘"
```

3요소를 명확히:

| 요소 | 설명 | 예시 |
|------|------|------|
| **데이터 범위** | 무엇을, 얼마만큼 | "120일 일봉", "20일 수급" |
| **분석 기준** | 어떤 방법으로 | "이동평균선 기준", "RSI로" |
| **원하는 결론** | 어떤 형태의 답 | "매수/관망 판단", "표로 정리" |

## 레벨 1: 기본 조회

```
삼성전자 현재가 알려줘
SK하이닉스 120일 일봉 데이터 보여줘
카카오 20일 외국인/기관 수급
현대차 PER이랑 PBR
코스피 지금 얼마야?
오이솔루션 종목코드 알려줘
```

## 레벨 2: 차트 분석

```
삼성전자 120일 일봉 보고 20일 이동평균선 기준으로 추세 판단해줘

SK하이닉스 주봉 60개 보고 지지선이랑 저항선 찾아줘

카카오 일봉 120일 보고 이동평균선(5/20/60/120일) 계산해서
정배열인지 역배열인지 알려줘

현대차 120일 차트 보고 볼린저밴드 계산해서
지금 과매수인지 과매도인지 판단해줘

네이버 일봉 60개 보고 RSI 14일 계산해줘.
과매도 구간이면 알려줘
```

## 레벨 3: 수급 분석

```
삼성전자 20일 수급 보고 외국인이 추세적으로 사는 건지 판단해줘

SK하이닉스 수급에서 기관이랑 외국인이 동시에 순매수한 날 찾아줘

카카오 수급 보고 개인이 팔고 외국인이 사는 구간이 있는지 확인해줘

현대차 수급 추이를 주가 흐름이랑 같이 분석해서
수급이 주가에 선행하는지 봐줘
```

## 레벨 4: 종목 비교

```
삼성전자랑 SK하이닉스 60일 차트 비교해서
어디가 상대적으로 강한지 분석해줘

카카오랑 네이버 차트 + 수급 + 재무 전부 비교해서 표로 정리해줘

현대차랑 기아 최근 수급 비교해서
외국인이 어디를 더 선호하는지 알려줘
```

## 레벨 5: 종합 리포트

```
삼성전자 차트 + 수급 + 재무 전부 조회해서
증권사 애널리스트처럼 종합 리포트 작성해줘.
매수/매도/관망 중 하나로 결론 내줘

SK하이닉스를 아래 항목으로 분석해줘:
1. 추세 (이동평균선 기반)
2. 모멘텀 (RSI, 거래량 변화)
3. 수급 (외국인/기관 방향성)
4. 밸류에이션 (PER/PBR)
5. 종합 의견
```

## 레벨 7: 복합 질문

```
SK하이닉스 120일 일봉 데이터 보고 20일 이동평균선 기준으로
현재 추세 판단하고, 외국인 20일 수급도 같이 봐서
지금 매수 진입하기 좋은 타이밍인지 판단해줘

삼성전자랑 SK하이닉스 최근 60일 차트 데이터 비교해서
어느 쪽이 상대적으로 강한지 분석해줘.
등락률, 거래량 변화, 수급 방향성 3가지 기준으로 표로 정리해줘

거래량 상위 30개 중에 외국인/기관이 동시에 순매수한 종목 3개 골라서
각각 차트 분석해줘

내가 삼성전자, 현대차, 카카오 보유 중인데
이 3종목의 최근 20일 수급 흐름이랑 재무지표 보고
지금 위험한 종목이 있는지 점검해줘
```

## 레벨 8: Excel 스냅샷 워크플로우

```
시가총액 상위 200개로 스냅샷 만들어줘

(위 질문 후)
그 스냅샷에서 PER 10 이하이고 낙폭 -20% 이상인 종목 찾아줘

(같은 파일)
그 중에서 외국인 순매수 상위 5개 보여줘

삼성전자 180일 차트 데이터 엑셀로 저장해줘
```

**팁**: 한 번 스냅샷 만들면 이후 쿼리는 **0.02초**로 즉시 답변.

## 프롬프트 작성 팁

### 1. 숫자를 명시하라
```
❌ "최근 차트 보여줘"
✅ "최근 120일 일봉 보여줘"
```

### 2. 분석 방법을 지정하라
```
❌ "추세 분석해줘"
✅ "20일 이동평균선 기준으로 추세 판단해줘"
```

### 3. 결론 형태를 요청하라
```
❌ "종합적으로 봐줘"
✅ "매수/관망/매도 중 하나로 결론 내줘"
```

### 4. 여러 도구 조합을 유도하라
```
"차트 + 수급 + 재무 종합해서"
"PER도 보고, 낙폭도 같이 봐줘"
```

### 5. Excel 스냅샷은 한 번 만들고 반복 활용하라
```
1회: "시총 100개 스냅샷 만들어줘"
반복: "그 파일에서 [조건] 찾아줘"
```
